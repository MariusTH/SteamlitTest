# A streamlit test

## Requirements
Python 3

## Dev

Setup local env :
```
python -m venv .venv

# Windows command prompt<
.venv\Scripts\activate.bat

# Windows PowerShell
.venv\Scripts\Activate.ps1

# macOS and Linux
source .venv/bin/activate
```

Install dependencies
```
pip install -r requirements.txt
```

Run app
`streamlit run app.py`
